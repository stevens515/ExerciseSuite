Ubuntu：/etc/hostname
Centos： /etc/sysconfig/network
添加用户组与用户
groupadd hadoop
useradd -d /home/hadoop -g hadoop hadoop
在Master节点上执行以下命令
ssh-keygen –t rsa –P ''
查看是否有.ssh文件
ls -a /home/hadoop
生成密钥
[hadoop@slave1 ~]$ ssh-keygen -t dsa -P '' -f ~/.ssh/id_dsa
把id_dsa.pub(公钥)追加到授权的key里面去:
cat ~/.ssh/id_dsa.pub >> ~/.ssh/authorized_keys
这段命令是把公钥加到用于认证的公钥文件中，这里的authorized_keys是用于认证的公钥文件。
至此元密码合建本机已经设置完毕。
验证
ssh localhost
rpm –qa | grep openssh
rpm –qa | grep rsync
否则联网通过yum install ssh和yum install rsync安装并启动：
service sshd start
